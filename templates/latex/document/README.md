# Short Note

A flat LaTeX template for short notes, memos, empirical appendices, and compact research writeups.

This version is intentionally lighter than the academic paper template: everything important lives close together so it is fast to copy, edit, and compile.

## Included files

- `note.tex`: the main note document
- `references.bib`: bibliography file
- `Makefile`: build commands
- `sample-note.pdf`: compiled sample output

## Included examples

- a regression-style equation
- a summary table
- a figure placeholder
- a proposition-style statement
- citations and footnotes

## Quick start

```bash
make
```

Then edit `note.tex` and `references.bib` as needed.
