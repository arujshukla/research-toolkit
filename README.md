# Research Toolkit

Reusable templates, snippets, and lightweight resources for research workflows.

This repository is meant to be a practical toolbox: things you (and myself) can copy into a project, adapt quickly, and use again later without rebuilding from scratch.

## What lives here

- `templates/`: copy-ready starters for slides, papers, handouts, and similar formats
- `snippets/`: small pieces of reusable code or markup
- `resources/`: reference material, and other supporting assets

## Current contents

- `templates/beamer/`: a warm-toned Beamer template built on `metropolis`

## Getting started

If you want to use one of the templates, copy the relevant folder into your project and edit from there rather than working directly in this repo.

## Notes

This is intentionally a living repository. Structure and docs can evolve as the toolkit grows.
