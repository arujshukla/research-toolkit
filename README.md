# Research Toolkit

Reusable templates, snippets, and lightweight resources for academic research workflows.

This repository is meant to be a practical toolbox: things you can copy into a project, adapt quickly, and use again later without rebuilding from scratch.

## What lives here

- `templates/`: copy-ready starters for slides, papers, handouts, and similar formats
- `snippets/`: small pieces of reusable code or markup
- `resources/`: reference material, checklists, and supporting assets

## Current contents

- `templates/beamer/warm-academic/`: a warm-toned Beamer template built on `metropolis`

## Design principles

- Reusable over bespoke
- Lightweight over overengineered
- Clear defaults with room for customization
- Academic presentation with practical usability

## Getting started

If you want to use one of the templates, copy the relevant folder into your project and edit from there rather than working directly in this repo.

For the Beamer template:

```bash
cd templates/beamer/warm-academic/template
make
```
